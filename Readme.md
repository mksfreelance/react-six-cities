# Проект "6 городов"
Данный проект создан с помощью [Create React App](https://github.com/facebook/create-react-app).

![](./public/img/screenshot.png)


#### Локальный запуск проекта
```bash
npm install
npm start
```

#### Сборка проекта
```bash
npm run build
```

#### Проверка линтером
```bash
npm run lint
```
